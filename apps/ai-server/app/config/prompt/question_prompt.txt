이 지원자의 정보에 기반해 총 5개의 면접 질문을 생성하세요.
                                                   
질문은 다음 기준에 따라 유형별로 구분되어야 합니다:
1. 인성 질문 (1개):  
   - 인재상 및 자소서 내용을 기반으로  
   - 성격, 동기, 가치관, 협업 태도 등을 평가할 수 있는 질문

2. 기술 질문 (2개):  
   - 사용자의 희망 직무(프론트엔드, 백엔드 등)를 기반으로  
   - 기술 역량, 도구 사용 능력, 개발 경험을 확인하는 질문

3. 맞춤 질문 (2개):  
   - 자소서와 포트폴리오, 희망 직무 정보를 모두 고려  
   - 특정 프로젝트, 기술 선택 이유, 문제 해결 사례에 대한 질문

또한, 각 질문은 아래 평가 항목 중 하나와 연관되어야 하며, 각 질문은 서로 다른 평가 항목을 골고루 다루도록 하세요.
해당 항목을 `criteria` 필드에 명시하세요:
- 태도/자세, 신뢰성, 적극성, 성실성/책임감, 협업능력, 창의성, 논리적 사고력, 기술 역량, 커뮤니케이션능력

        "question_type": "기술",
        "criteria": "기술 역량"
    }},
    ...
]
                                                   
각 질문은 실제 면접에서 사용할 수 있도록 명확하고 구체적으로 작성해주세요.
중복된 질문 없이, 유형/내용/평가 항목이 명확히 분리된 5개의 질문을 생성하세요.
